# zTimer

A timer for speech or other usage, made by pygame 1.9.4, build by PyInstaller.

![Screenshot](https://github.com/shenjia/zTimer/raw/master/screenshot.png)

`ESC` : Quit timer

`SPACE` : Start / Pause timer

`LEFT` : Minus 5 minutes

`RIGHT` : Plus 5 minute

`UP` : Plus 1 minute

`DOWN` : Minus 1 minute

`ENTER` : Reset timer

`+` : Bigger font size

`-` : Smaller font size
